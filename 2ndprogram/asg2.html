COSC 2331 Assignment #2

Due Date: Friday, September 11, 2009 by 5pm

Your assignment is to write a NASM program that prompts the user to enter a string of up to 70 chars. It should echo the original string back to the user, then capitalize all letters, echo it back, then lower case all letters and echo it back. Be sure your program outputs appropriate messages, spaces, and newline chars as the output is produced. You will write your entire program in assembly language so you will assemble it, link it, and then execute it.

Example for creating this program and running it

  1. Create prog2.asm using your editor:    emacs prog2.asm
  2. Assemble prog2.asm using nasm:    nasm -f elf prog2.asm
  3. Link your prog2.o:    ld -m elf_i386 prog2.o
  4. Execute your program:    ./a.out
